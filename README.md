# dtpa
Examples and answer of exercises for the paper "Dependently typed programming in Agda"
http://www.cse.chalmers.se/~ulfn/papers/afp08/tutorial.pdf
* This paper is still a best introduction for Agda beginner, but some of codes are out of date. Then I rewrite codes in accordance with current Agda standard library (stdlib-0.9 or later).
